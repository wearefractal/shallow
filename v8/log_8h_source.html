00039 // Logger is used for collecting logging information from V8 during
00040 // execution. The result is dumped to a file.
00041 //
00042 // Available command line flags:
00043 //
00044 //  --log
00045 // Minimal logging (no API, code, or GC sample events), default is off.
00046 //
00047 // --log-all
00048 // Log all events to the file, default is off.  This is the same as combining
00049 // --log-api, --log-code, --log-gc, and --log-regexp.
00050 //
00051 // --log-api
00052 // Log API events to the logfile, default is off.  --log-api implies --log.
00053 //
00054 // --log-code
00055 // Log code (create, move, and delete) events to the logfile, default is off.
00056 // --log-code implies --log.
00057 //
00058 // --log-gc
00059 // Log GC heap samples after each GC that can be processed by hp2ps, default
00060 // is off.  --log-gc implies --log.
00061 //
00062 // --log-regexp
00063 // Log creation and use of regular expressions, Default is off.
00064 // --log-regexp implies --log.
00065 //
00066 // --logfile <filename>
00067 // Specify the name of the logfile, default is "v8.log".
00068 //
00069 // --prof
00070 // Collect statistical profiling information (ticks), default is off.  The
00071 // tick profiler requires code events, so --prof implies --log-code.
