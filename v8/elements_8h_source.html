00048   // Returns true if a holder contains an element with the specified key

00068   // Modifies the length data property as specified for JSArrays and resizes the
00069   // underlying backing store accordingly. The method honors the semantics of
00070   // changing array sizes as defined in EcmaScript 5.1 15.4.5.2, i.e. array that
00071   // have non-deletable elements can only be shrunk to the size of highest
00072   // element that is non-deletable.
00073   virtual MaybeObject* SetLength(JSArray* holder,
00074                                  Object* new_length) = 0;

00076   // Modifies both the length and capacity of a JSArray, resizing the underlying

00079   // elements. This method should only be called for array expansion OR by

00091   // If kCopyToEnd is specified as the copy_size to CopyElements, it copies all
